# NYTimes Demo iOS #

_NYTimes_ for iOS (Apple iPhone)

# Getting Setup #

* Install Xcode
* Install Cocoapods
* Run `$ pod install`from the Xcode_project directory.
* Open NYtimes.xcworkspace in Xcode

# Fastlane #

* Install fastlane
* Run `$ fastlane tests` from the Xcode_project directory to run UI and Unit tests.

# Requirements: #
- Xcode 10.0
- CocoaPods
