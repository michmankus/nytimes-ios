# NYTimes Demo iOS #

_NYTimes_ for iOS (Apple iPhone)

# Getting Setup #

* Install Xcode
* Install Cocoapods
* Run `$ pod install`from the Xcode_project directory.
* Open NYtimes.xcworkspace in Xcode

# Requirements: #
- Xcode 10.0
- CocoaPods
